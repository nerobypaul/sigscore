# Headless CRM - Frontend

React-based admin UI for the Headless CRM system.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + TypeScript
- Vite
- TailwindCSS
- TanStack Query
- React Router

## Development

The Vite dev server proxies `/api` requests to the backend at `http://localhost:3001`.
